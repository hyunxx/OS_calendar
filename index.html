User Guide

대학생의 생활 습관 및 일정관리를 도와주는 어플로, 네 가지의 기능을 갖고 있습니다.

1. 캘린더 📅
-월간 / 주간 / 시간별로 일정 등록이 가능합니다. 다크모드를 지원합니다.

2. 타이머 ⏰
-대학생들의 공부를 위한 타이머 기능으로, 최대 59분의 시간 설정이 가능합니다.

3. To-Do List ✅
-날짜와 할 일을 설정하면, 날짜 별로 리스트 형태로 정렬됩니다. 할 일을 한눈에 보고 관리할 수 있습니다.

4. D-Day 설정 📝
-날짜를 등록하면 D-Day를 리스트에 추가해 보여줍니다. 시험 날짜, 중요 일정 등을 체크할 수 있습니다.

캘린더와 투두리스트, 디데이는 등록 내용이 저장됩니다.

Developer Guide

프로젝트의 구성은 html / css / vanila js로 구성됩니다.

공통 파일은 header / nav_bottom이며 각 페이지의 script 태그를 통해 공통으로 사용됩니다.

각 페이지에서 사용자가 작성한 내용은 localstorage에 저장됩니다.

calendar 페이지의 경우 오픈소스인 fullCalendar를 사용하며, 사용 옵션은 페이지의 script를 통해 참조할 수 있습니다.
